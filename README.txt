You do not need to install any packages or dependencies. To run
the site, navigate to index.html and open it in your browser of
choice. You can use live server if you want, but that's not ess-
ential. You can see every part of the site if you start from the
index. Javascript needs to be enabled.

The mobile site and the desktop are the exact same site. The site
has been built to work at any width from 350px up.
